# OnlineCalculatePractice
一个纯前端技术栈实现的在线计算题练习页面

这个项目是课程作业，用了ES6语法和面向对象技术。

[示范地址](http://zhpjy.github.io/OnlineCalculatePractice/index.html)

### 功能

- 用户可以选择加法、减法和混合三种算式

- 可以输入设定算式的数量、算式的操作数范围、算式的结果范围

- 在线生成试题，在线做题，在线批改，显示批改结果

- 可以在生成试题之后下载 PDF 试题
